# Features

- [x] Babel
- [x] Cross-env
- [x] Eslint & Prettier
- [x] Sass
- [x] Axios pre-installed
- [ ] TypeScript support

# Usage

## Development

- Clone the repo.

```shell
$ git clone https://github.com/redgroot/vue-parcel-boilerplate.git project-name
```

- Install dependencies.

```shell
$ npm install
```

- Run development server

```shell
$ npm run dev
```

## Production

```shell
$ npm run build
```
